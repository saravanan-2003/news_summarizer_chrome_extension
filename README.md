# SummarizeMe Extension

SummarizeMe Extension is a browser extension that allows users to summarize news articles directly in their browser. It provides a convenient way to get quick insights into news articles without having to read them in full.

## Features

- Summarize news articles with a single click.
- View summarized content directly within the extension popup.
- Supports summarization of articles in multiple languages.

## Installation

### Google Chrome

1. Download the extension ZIP file from the [releases page](#).
2. Unzip the downloaded file.
3. Open Google Chrome and navigate to `chrome://extensions`.
4. Enable Developer mode (toggle switch in the top right corner).
5. Click on the "Load unpacked" button.
6. Select the unzipped extension folder and click "Open".

## Usage

1. Click on the extension icon in the browser toolbar to open the popup.
2. Navigate to a news article you want to summarize.
3. Click the "Summarize" button in the popup.
4. The summarized content will be displayed in the popup.
